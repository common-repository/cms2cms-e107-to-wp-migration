Plugin for automated data migration from e107 to WordPress. Convert e107 to WordPress easily and fast.

== Description ==
This plugin will help you to move all website content from e107 to WordPress swiftly and accurately in a few simple steps with no coding skills necessary.

= HOW THIS PLUGIN HELPS: =
- There is Free Demo available (migration of limited number of content);
- Complete Migration price depends on the content amount you want to move to the New website;
- Data mapping option allows you to choose what content to move to WordPress;
- Migration time depends on the size of data to be migrated.

= WHAT WILL BE MOVED FROM e107 TO WORDPRESS? =
* Pages along with the content included there;
* Attachments (images and videos);
* Images included in the content body.


= PRESERVED SEO JUICE AFTER THE MIGRATION =
* Migration of SEO friendly URLs;
* Migration of keywords, metadata/meta description;
* Automated 301 redirects from existing e107 website to the new WordPress website.


= COMPATIBLE WITH: WORDPRESS 4.1.X – 5.0.X. =
- Our support team is available live 24/7.
- For more customized migration type, CMS2CMS offers data migration service packages to order.


== Installation ==
1. Download the plugin zip file.
2. Go to the Admin Panel -> Plugins -> Add New -> Upload Plugin.
3. Choose the downloaded file in zip format and click the “Install Now” button.
4. Go to Admin -> Plugins, activate the “CMS2CMS: Automated e107 To WordPress Migration Plugin”.
5. Register an account or sign in to the existing one.
6. Provide your Existing website URL address.
7. Click ‘Verify Connection’. You’ll be redirected to the CMS2CMS website to proceed with your migration.
